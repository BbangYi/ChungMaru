# 청마루 HTML-to-PPTX v2

## 최종 파일

- `chungmaru_html_to_pptx_v2.pptx`
- `chungmaru_html_to_pptx_v2.html`
- `chungmaru_html_to_pptx_v2_contact_sheet.png`

## 생성 방식

이번 버전은 PowerPoint 도형을 직접 조립한 v1과 다르게, HTML/CSS를 먼저 만들고 각 슬라이드를 브라우저에서 2배 해상도 PNG로 렌더링한 뒤 PPTX에 삽입했다.

장점:

- HTML/CSS 기준이라 시각 품질과 레이아웃 통제가 더 좋다.
- 서비스 정의서의 문제 정의, 사용자, 시나리오, 차별점, 범위 정의까지 반영해 내용 밀도가 높다.
- 16장 구성으로 최초 계획, 현재 개발 상황, 완료 산출물, 모델 평가, 제약, 향후 계획이 더 자연스럽게 이어진다.

한계:

- PPTX 내부 텍스트는 이미지로 들어가므로 PowerPoint에서 문장 단위 편집은 어렵다.
- 내용 수정은 `chungmaru_html_to_pptx_v2.html`을 고친 뒤 재렌더링하는 방식이 가장 안전하다.

6. 범위 정의

9. 개발 진행상황
10. 완료된 산출물
11. 시스템 구조
12. Demo 기준 기능
13. 모델 평가 결과
14. 실패 유형 분석
15. 제약 사항
