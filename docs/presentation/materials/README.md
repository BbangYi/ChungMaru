# 발표 근거 자료

이 폴더는 발표 자료 작성에 필요한 요약 근거만 보관합니다.
원본 로그, 대용량 JSON, 이미지 차트, PDF/PPTX는 재생성 가능한 산출물로 보고 Git 추적에서 제외합니다.

## 포함 자료

| 파일 | 내용 |
| --- | --- |
| `project-context-summary.md` | 서비스 정의, 제약, Android 실시간 탐지 방향 요약 |
| `backend-model-failure-summary-20260506.md` | backend 모델 실패 분석 핵심 수치와 해석 |
| `parser-extractor-evidence-plan-20260514.md` | 파서/추출기 증빙 계획, Android 앱/검색 결과 영역 수집 가능성, raw/cleaned JSONL 비교 기준 |

## 사용 규칙

- 발표나 보고서 문장은 가능한 한 이 폴더의 요약 자료에서 가져옵니다.
- 새로운 실험 결과가 생기면 원본 산출물을 그대로 넣기보다 핵심 수치, 실패 원인, 검증 명령, 남은 리스크만 추가합니다.
- raw 데이터가 꼭 필요할 때는 경로만 기록하고, Git 추적 대상에 넣지 않습니다.
